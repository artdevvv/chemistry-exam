Электролитическая диссоциация

h2>

Электролитическая диссоциация — это процесс распада молекул электролита на ионы (положительные и отрицательные частицы) при растворении их в воде или другом растворителе. Этот процесс возможен только для веществ, которые называются электролитами.

Неэлектролиты и электролиты

  • Электролиты — это вещества, которые в растворе диссоциируют на ионы и проводят электрический ток. Пример: NaCl (хлорид натрия) в воде:

    NaCl → Na+ + Cl-

  • Неэлектролиты — это вещества, которые не диссоциируют на ионы в растворе и не проводят электрический ток. Пример: сахар (C₆H₁₂O₆) в воде.

Степень диссоциации

Степень диссоциации (\(\alpha\)) — это доля молекул вещества, которые распались на ионы в растворе, по отношению к общему числу молекул вещества. Для слабых электролитов степень диссоциации зависит от концентрации раствора.

Формула для степени диссоциации:

\(\alpha = \frac{\text{число молекул, распавшихся на ионы}}{\text{общее число молекул вещества в растворе}}\)

Сильные и слабые электролиты

  1. Сильные электролиты — вещества, которые полностью диссоциируют на ионы в растворе. Примеры: NaCl, H₂SO₄ (серная кислота), NaOH (гидроксид натрия).

    Пример уравнения диссоциации для сильного электролита:

    NaCl → Na+ + Cl-

    Для сильных электролитов степень диссоциации близка к 1, так как они почти полностью распадаются на ионы.

  2. Слабые электролиты — вещества, которые частично диссоциируют в растворе. Примеры: CH₃COOH (уксусная кислота), NH₃ (аммиак).

    Пример уравнения диссоциации для слабого электролита:

    CH₃COOH ↔ CH₃COO- + H+

    Для слабых электролитов степень диссоциации меньше 1, и она зависит от концентрации раствора.

Константа диссоциации

Константа диссоциации (\(K_d\)) — это величина, которая показывает, насколько сильно вещество диссоциирует в растворе. Для слабых электролитов она выражается через концентрации ионов и исходных молекул. Для кислоты или основания:

\(K_d = \frac{[\text{ион}_1][\text{ион}_2]}{[\text{молекула}_0]}\)

Пример для уксусной кислоты (CH₃COOH):

\(K_d = \frac{[CH₃COO^-][H^+]}{[CH₃COOH]}\)

Примеры диссоциации слабых кислот

  1. Муравьиная кислота (HCOOH):

    HCOOH ↔ H+ + HCOO-

    Константа диссоциации для муравьиной кислоты:

    \(K_d = \frac{[H^+][HCOO^-]}{[HCOOH]}\)

  2. Сероводородная кислота (H₂S):

    Сероводородная кислота диссоциирует в два этапа:

    H₂S ↔ H+ + HS- (первый этап)

    HS- ↔ H+ + S2- (второй этап)

    Константы диссоциации для этих этапов:

    \(K_1 = \frac{[H^+][HS^-]}{[H₂S]}, \quad K_2 = \frac{[H^+][S^{2-}]}{[HS^-]}\)

Пример диссоциации слабого основания

Гидроксид аммония (NH₄OH):

NH₄OH ↔ NH₄+ + OH-

Константа диссоциации для аммония:

\(K_b = \frac{[NH₄^+][OH^-]}{[NH₄OH]}\)

Закон разбавления Оствальда

Закон разбавления Оствальда описывает, как степень диссоциации изменяется при разбавлении раствора слабой кислоты или основания. С увеличением объема (разбавлением) раствор слабой кислоты или основания диссоциирует больше.

Для слабой кислоты закон Оствальда:

\(\alpha = \sqrt{\frac{K_d}{C}}\)

где \(C\) — концентрация кислоты, \(\alpha\) — степень диссоциации, \(K_d\) — константа диссоциации.

Для слабого основания:

\(\alpha = \sqrt{\frac{K_b}{C}}\)

где \(K_b\) — константа диссоциации основания, \(C\) — концентрация основания.
