24. Энергия Гиббса. Стандартная энергия Гиббса образования. Свободная энергия Гиббса реакций. Эксергонические и эндергонические процессы, протекающие в организме (примеры). Принцип энергетического сопряжения.

1. Энергия Гиббса

Энергия Гиббса (G) — это мера энергии, которую система может использовать для совершения работы при постоянных температуре и давлении.

Простыми словами:

  • Если энергия Гиббса уменьшается (ΔG < 0) — процесс идёт самопроизвольно.
  • Если увеличивается (ΔG > 0) — процесс невозможен без внешней энергии.

Формула энергии Гиббса:

G = H - TS

Где:

  • G — энергия Гиббса,
  • H — энтальпия (тепловая энергия),
  • T — температура в К,
  • S — энтропия (мера беспорядка).

Вывод: Процесс будет идти сам, если компенсируется потеря энергии или рост беспорядка.

2. Стандартная энергия Гиббса образования

Стандартная свободная энергия Гиббса образования (ΔGf0) — это изменение энергии Гиббса при образовании 1 моль вещества из простых веществ при стандартных условиях:

  • Давление 1 атм
  • Температура 298 К (25°C)

Примеры стандартной энергии Гиббса образования:

  • ΔGf0 (O2) = 0 (простые вещества в стандартных условиях имеют ΔGf0 = 0).
  • ΔGf0 (H2O) = -237 kJ/mol (вода образуется самопроизвольно).
  • ΔGf0 (CO2) = -394 kJ/mol (углекислый газ легко образуется при горении углерода).

Вывод: Чем ниже ΔGf0, тем легче вещество образуется.

3. Свободная энергия Гиббса реакции

Как понять, пойдёт ли реакция сама?

Для любой реакции рассчитывают изменение свободной энергии Гиббса:

ΔG = ΔH - T ΔS

Где:

  • ΔH — теплота реакции (энтальпия),
  • T — температура в К,
  • ΔS — изменение беспорядка (энтропии).

Как интерпретировать ΔG?

Значение ΔG Вывод Пример
ΔG < 0 Реакция идёт сама (экзергоническая) Горение топлива, гидролиз АТФ
ΔG > 0 Реакция не идёт сама (эндергоническая) Синтез белка, фотосинтез
ΔG = 0 Равновесие, система стабильна Вода в закрытом сосуде

Вывод: Реакция идёт сама, если ΔG < 0.

4. Экзергонические и эндергонические процессы в организме

А) Экзергонические процессы (ΔG < 0)

Эти процессы выделяют энергию и идут самопроизвольно.

Примеры в организме:

  • Гидролиз АТФ:
    ATP + H2O → ADP + Pi + энергия
    (даёт энергию для работы клетки, ΔG = -30,5 кДж/моль)
  • Расщепление глюкозы (гликолиз):
    C6H12O6 + 6O2 → 6CO2 + 6H2O + энергия
    (используется для получения энергии, ΔG ≈ -2870 кДж/моль)

Б) Эндергонические процессы (ΔG > 0)

Эти процессы требуют энергию и не идут сами.

Примеры в организме:

  • Синтез белков из аминокислот:
    Аминокислоты → Белок
    (требует энергии, полученной из АТФ)
  • Фотосинтез:
    CO2 + H2O + свет → C6H12O6 + O2
    (нужен солнечный свет для реакции)

Вывод: Организм использует экзергонические реакции, чтобы запустить эндергонические.

5. Принцип энергетического сопряжения

Так как многие реакции в клетке требуют энергии (ΔG > 0), организм связывает их с реакциями, которые дают энергию (ΔG < 0).

Как это работает?

  • Реакции, которые дают энергию (гидролиз АТФ), используются для запуска реакций, которые требуют энергию (синтез белка).
  • Это работает как зарядка телефона:
    • Розетка даёт энергию (экзергоническая реакция).
    • Телефон использует её для зарядки аккумулятора (эндергоническая реакция).

Пример сопряжения в клетке:

  • Гидролиз АТФ даёт энергию (ΔG < 0):
    ATP + H2O → ADP + Pi + энергия (-30,5 кДж/моль)
  • Синтез белка требует энергию (ΔG > 0)
  • Клетка соединяет эти реакции, используя энергию гидролиза АТФ.

Вывод: Организм связывает реакции так, чтобы энергия отдающих процессов использовалась для требующих её процессов.

Выводы

  • Энергия Гиббса показывает, возможен ли процесс сам по себе.
  • Реакции с ΔG < 0 идут сами (экзергонические), а с ΔG > 0 требуют энергию (эндергонические).
  • Организм использует принцип энергетического сопряжения: энергия гидролиза АТФ идёт на другие процессы.

Главный принцип:

🌍 Энергия в клетке перераспределяется, как деньги в экономике — одни процессы её зарабатывают, другие тратят! 🚀
